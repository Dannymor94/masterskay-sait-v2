# Мастерская — новый сайт (custom, design-first)

Проектный пакет под разработку в Claude Code. Центр практик «Мастерская»,
Ростов-на-Дону. Кастомный сайт по бренд-буку; главная задача V1 — **красивый
современный дизайн + загрузка залов через аренду**.

## С чего начать (для Claude Code)
1. Прочитать `PROJECT_GUIDE.md` — каноничное описание (главный документ).
2. `DECISIONS.md` — что уже решено (не пере-решать молча) и что открыто.
3. `SPEC.md` — что строим (лендинг аренды — герой).
4. `CLAUDE.md` — инварианты + **правила вызова агентов** + runbook M0.
5. `PLAN.md` — стадии TDD; старт с **M0** (каркас + дизайн-система + лендинг
   аренды на плейсхолдерах).

## Ключевые решения (зафиксированы)
- Позиционирование — **гибрид**: тёплое ядро бренд-бука + весь спектр услуг честно,
  без эзотерического тумана.
- Платформа — **custom** (стек по умолчанию Astro+SQLite, см. PROJECT_GUIDE §6).
- Главная конверсия V1 — **заявка на аренду**. Визит/сообщество/SEO — вторично/позже.
- Команда — Даниил + ассистент + Claude Code, TDD.

## Что нужно от Даниила, чтобы доделать (см. PROJECT_GUIDE §10)
Ставки аренды по трём залам · кто уже арендует (блок доверия) · есть ли фото залов
или нужна съёмка (гейтит визуал) · правило приоритета вечернего слота · выгрузка
Метрики 3–6 мес.

## Внешняя зависимость
**Фотограф** — критичная, ранняя, единственная заведомо платная позиция (живой
человек). До съёмки дизайн собирается на качественных плейсхолдерах.
